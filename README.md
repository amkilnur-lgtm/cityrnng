# CITYRNNG

Monorepo for the CITYRNNG platform.

## Stack

- pnpm workspaces
- Turborepo
- Next.js
- NestJS
- PostgreSQL
- Redis

## Workspaces

- apps/web
- apps/api
- packages/ui
- packages/types
- packages/config

## Scripts

- pnpm install
- pnpm -r build
- pnpm -r lint
- pnpm -r test
- pnpm -r typecheck
